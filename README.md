Apuntes del trabajo.

Adaptativo a plataformas, IOS, APK
REDE SOCIAL (alta prioridad)
La idea es gestionar unos juegos de mesa
Entras con usuario, inicio de sección, puedes entrar como invitado
Al entrar tiene una parte de muro de redes sociales donde puede ver la comunidad jugando, ver fotos, crear chat, interacciones, como roblox o como una red social.
Un perfil con perfiles enlazados (amigos, y un muro donde aparezca cosas de los amigos)
Clubs de juegos, como una comunidad entera, clans que tengan publicaciones e interacciones personales.
Publicidad en medio del muro de publicaciones
Publicaciones de rango, conquista, he metido una paliza a pepe
Las personas tienen datos como ubicación para hacer quedadas. (se puede habilitar o no)
A que juego está jugando.
Reconocer jugadores cerca para jugar.
Elo de cada juego, ranking.


GESTION DE PARTIDAS (media prioridad)

Juegos de mesa en físico.
Gestión de partidas sueltas
Gestión de torneos 
Gestion de Ligas(1 dia o 2 de torneo para apuntarse, se apunta mucha gente y juegan sus partidas con puntuaciones y un ganador)
Gestión de pagos del torneo, rankings, publis, patrocinios
Roles para los torneos y para la app
Moderadores y gestión de torneo

SIMULADOR DE MESA (menor prioridad) 
Table top simulator
